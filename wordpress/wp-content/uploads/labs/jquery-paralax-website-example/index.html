jQuery paralax plugin website example

This is a paralax plugin I wrote for a site called battleback1000.co.uk. I also used it on my homepage.

Its easy to set up and works both horizontally and vertically.

The paralax elements speed and position can be set using data-top and data-left attributes. They dont have to be absolut positioned like other paralax plugins. Just style them naturally and the plugin will do the rest.

Just click a nav item above to see the plugin in action. Ps. The bg images are pretty large so if you see red it means they havn't loaded yet.
